# 이슈 관리 규칙
| 단계 | 수행 방법 |
| --- | --- |
| 1단계 | 이슈 기록 : 이슈 관리 대장에 이슈 당사자가 기록합니다. |
| 2단계 | 혼자 고민 : 혼자 30분간 해결 방법을 찾습니다. |
| 3단계 | 페어 프로그래밍 : 페어 프로그래밍 파트너 혹은 이슈 관련 담당자와 논의합니다. |
| 4단계 | 팀원과 함께 고민 : 프로젝트 리더에게 이슈 보고, 프로젝트 리더가 이슈 관리 회의 진행해 팀원 전체가 해결 방법을 논의해 탐색합니다. |
| 5단계 | 해결 방법 기록 : 이슈가 해결되면 이슈 관리 대장에 이슈 당사자가 해결 방법 기록합니다. |
